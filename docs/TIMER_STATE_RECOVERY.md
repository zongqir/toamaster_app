# 计时状态恢复功能说明

## 功能概述

启航AACTP 时间官现在支持**计时状态恢复**功能，允许用户在暂停后切换环节，再回来时可以继续计时，已用时间会自动累加。

## 使用场景

### 场景 1：临时切换环节
1. 正在为"备稿演讲"计时（已用时 3 分钟）
2. 点击"暂停"
3. 点击"下一项"切换到"备稿点评"
4. 处理完点评后，点击"上一项"回到"备稿演讲"
5. **结果**：计时器显示 3 分钟（之前的已用时间），可以继续计时

### 场景 2：通过 Agenda 快速跳转
1. 正在为第 5 个环节计时（已用时 2 分钟）
2. 点击"暂停"
3. 点击"查看 Agenda 日程"，选择第 8 个环节
4. 完成第 8 个环节后，再次打开 Agenda，回到第 5 个环节
5. **结果**：计时器显示 2 分钟（之前的已用时间），可以继续计时

### 场景 3：意外切换后恢复
1. 正在计时，不小心点击了"下一项"
2. 立即点击"上一项"回到原环节
3. **结果**：已用时间保留，可以继续计时

## 技术实现

### 自动保存机制
- 切换环节时，自动保存当前环节的 `actualDuration`（实际用时）
- 保存 `actualEndTime`（结束时间）
- 数据存储在会议 session 中

### 自动恢复机制
- 切换到新环节时，检查该环节是否已有计时记录
- **情况 1**：如果有 `actualDuration`，直接恢复该时间
- **情况 2**：如果有 `actualStartTime` 但没有 `actualEndTime`，计算已用时间（当前时间 - 开始时间）
- **情况 3**：如果没有任何记录，从 0 开始

### 重置功能
- 点击"重置"按钮（长按当前环节标题）
- 清除 `actualDuration`、`actualStartTime`、`actualEndTime`
- 计时器归零，可以重新开始

## 用户界面

### 计时页面
- **开始/暂停按钮**：控制计时状态
- **上一项/下一项按钮**：切换环节（自动暂停并保存）
- **查看 Agenda 日程**：快速跳转到任意环节（自动暂停并保存）
- **重置按钮**：长按当前环节标题，清除计时记录

### 视觉反馈
- 已计时的环节：显示绿色勾选标记
- 当前环节：蓝色高亮显示
- 待计时的环节：普通样式

## 注意事项

1. **自动暂停**：切换环节时，如果正在计时，会自动暂停
2. **数据保存**：切换时自动保存当前进度，无需手动操作
3. **时间累加**：多次计时的时间会累加（除非重置）
4. **重置操作**：只有明确点击"重置"按钮才会清零

## 常见问题

### Q: 切换环节后，时间会丢失吗？
A: 不会。切换时会自动保存当前已用时间，回来时会自动恢复。

### Q: 如何清零某个环节的计时？
A: 长按该环节的标题，点击"重置"按钮。

### Q: 如果不小心点了"下一项"怎么办？
A: 立即点击"上一项"回到原环节，已用时间会保留。

### Q: 可以在多个环节之间来回切换吗？
A: 可以。每个环节的计时状态都是独立保存的。

### Q: 会议结束后，计时记录会保存吗？
A: 会。所有计时记录都会保存在会议历史中，可以随时查看统计报表。

## 最佳实践

1. **正常流程**：按顺序计时，使用"下一项"按钮切换
2. **临时调整**：需要临时处理其他环节时，先暂停，切换过去，完成后再切回来
3. **快速跳转**：使用 Agenda 日程列表快速跳转到任意环节
4. **重新计时**：如果需要重新计时某个环节，使用"重置"功能
