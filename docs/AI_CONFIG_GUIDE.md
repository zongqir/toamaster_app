# AI 配置指南

## 概述

启航AACTP 时间官支持 AI 智能解析会议流程表格。系统**默认使用文心AI**（百度文心一言大模型），无需任何配置即可使用。如果您希望使用其他 AI 服务（如硅基流动、DeepSeek 等），可以在设置页面配置自定义 AI。

## 默认 AI 服务

### 文心AI（推荐）

- **无需配置**：开箱即用
- **服务提供商**：百度文心一言
- **适用场景**：所有用户，特别是不想配置 API 的用户
- **优势**：
  - 免配置，立即可用
  - 稳定可靠
  - 中文理解能力强
  - 适合解析中文会议流程

## 自定义 AI 服务

如果您希望使用其他 AI 服务，可以在设置页面开启"使用自定义 AI"开关，并填写以下信息：

### 配置步骤

1. 打开小程序，进入"设置"页面
2. 找到"AI 解析配置"区域
3. 开启"使用自定义 AI"开关
4. 填写以下信息：
   - **API 地址**（必填）：AI 服务的 API 端点
   - **API Key**（必填）：您的 API 密钥
   - **模型名称**（可选）：指定使用的模型

### 推荐服务商

#### 1. 硅基流动（SiliconFlow）

- **官网**：https://siliconflow.cn
- **特点**：
  - 性价比高
  - 注册即送免费额度
  - 支持多种开源大模型
- **配置示例**：
  - API 地址：`https://api.siliconflow.cn/v1/chat/completions`
  - 模型名称：`Qwen/Qwen2.5-7B-Instruct`（推荐）

#### 2. DeepSeek

- **官网**：https://platform.deepseek.com
- **特点**：
  - 国产大模型
  - 推理能力强
  - 价格实惠
- **配置示例**：
  - API 地址：`https://api.deepseek.com/v1/chat/completions`
  - 模型名称：`deepseek-chat`

#### 3. 其他兼容 OpenAI 格式的 API

只要 API 兼容 OpenAI 的 `/v1/chat/completions` 格式，都可以使用。

### 配置说明

- **API Key 安全**：您的 API Key 仅存储在本地设备，不会上传到服务器
- **切换服务**：可以随时关闭"使用自定义 AI"开关，恢复使用默认的文心AI
- **模型名称**：如果不填写，系统会使用 API 服务商的默认模型

## 使用方式

### 导入页面

1. 在导入页面粘贴会议流程表格
2. 点击右上角的"AI 解析"标签（默认开启）
3. 点击"智能解析"按钮
4. 系统会自动使用配置的 AI 服务（默认文心AI）进行解析

### 本地解析

如果不想使用 AI 解析，可以：

1. 点击右上角的"本地解析"标签
2. 点击"智能解析"按钮
3. 系统会使用本地规则进行解析（不需要网络请求）

## 常见问题

### Q: 默认的文心AI需要配置吗？
A: 不需要。文心AI是默认服务，无需任何配置即可使用。

### Q: 如何获取 API Key？
A: 访问对应服务商的官网（如硅基流动、DeepSeek），注册账号后在控制台获取 API Key。

### Q: API Key 会被上传到服务器吗？
A: 不会。API Key 仅存储在您的本地设备，不会上传到任何服务器。

### Q: 可以随时切换 AI 服务吗？
A: 可以。在设置页面关闭"使用自定义 AI"开关即可恢复使用默认的文心AI。

### Q: 自定义 AI 解析失败怎么办？
A: 请检查：
1. API 地址是否正确
2. API Key 是否有效
3. 账户是否有足够的额度
4. 如果仍然失败，可以切换回默认的文心AI或使用本地解析

### Q: 本地解析和 AI 解析有什么区别？
A: 
- **本地解析**：使用预设规则，速度快，但可能无法处理复杂格式
- **AI 解析**：使用大模型智能理解，准确率高，能处理各种格式

## 技术细节

### API 格式要求

自定义 AI 服务必须兼容 OpenAI 的 Chat Completions API 格式：

**请求格式**：
```json
{
  "messages": [
    {"role": "system", "content": "系统提示词"},
    {"role": "user", "content": "用户输入"}
  ],
  "stream": false
}
```

**响应格式**：
```json
{
  "choices": [
    {
      "message": {
        "content": "AI 回复内容"
      }
    }
  ]
}
```

### 认证方式

- **文心AI**：使用集成密钥（自动注入）
- **自定义AI**：使用 `Authorization: Bearer YOUR_API_KEY` 头部
