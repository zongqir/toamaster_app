# 启航AACTP 时间官

一个专为 Toastmasters 会议设计的计时管理小程序，支持导入会议流程、AI 智能解析、实时计时控制、会议统计分析等完整功能。

## ✨ 主要功能

### 📋 会议管理
- 导入会议流程表格（支持 TSV、PDF 复制文本）
- AI 智能解析会议信息（自动识别环节、负责人、时长）
- 会议历史记录
- 复制会议模板
- 删除和重置会议

### ⏱️ 计时功能
- 大字体倒计时显示
- 自动黄牌/红牌提示（震动 + 背景变色）
- 时间微调（±10s、±30s）
- 长按重置计时
- 实时编辑当前环节
- 临时新增/删除环节
- 查看剩余环节列表

### 📊 统计分析
- 整体时长对比
- 按负责人分组统计
- 超时/准时/提前分析
- 详细环节记录
- 一键复制复盘数据

### ⚙️ 规则设置
- 自定义黄牌/红牌阈值
- 短时/长时规则配置
- 环节类型默认设置
- 成员名单管理

## 🚀 快速开始

### 前置要求

- Node.js 18.x 或 20.x（推荐）
- pnpm 包管理器（项目使用 pnpm，**不要用 npm/yarn**）

安装 pnpm（如果尚未安装）：
```bash
npm install -g pnpm
```

### 方法 1：编译为微信小程序（推荐）

```bash
# 1. 安装依赖（首次运行）
pnpm install

# 2. 编译小程序
pnpm run build:weapp

# 或使用快速脚本
./build-weapp.sh
```

编译完成后：
1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具
3. 选择"导入项目"
4. 项目路径选择：`项目根目录/dist`
5. AppID 选择"测试号"（如果没有正式 AppID）
6. 点击"导入"即可预览

### 方法 2：编译为 H5 网页版

```bash
# 编译 H5 版本
pnpm run build:h5

# 开发模式（支持热更新）
pnpm run dev:h5
```

编译后的文件在 `dist` 目录，可以部署到任何静态网站托管服务。

## 🛠️ 技术栈

- **框架**：Taro 4.1.10

## 📖 使用说明

### 1. 导入会议流程

1. 从 PDF 或 Excel 复制会议流程表格
2. 粘贴到导入页面
3. 选择"AI 智能解析"或"本地快速解析"
4. 预览解析结果
5. 点击"确认并继续"

### 2. 编辑时间轴

1. 查看解析后的会议流程
2. 可以编辑环节名称、负责人、时长
3. 可以新增、删除、禁用环节
4. 点击"保存并开始计时"

### 3. 开始计时

1. 点击"开始"按钮开始计时
2. 倒计时显示剩余时间
3. 黄牌/红牌自动提示
4. 可以随时暂停、调整时间
5. 长按"暂停"按钮可重置计时
6. 点击"下一项"进入下一个环节

### 4. 查看统计

1. 会议结束后自动保存记录
2. 在历史页面选择已完成的会议
3. 点击"统计"图标查看详细数据
4. 可以复制复盘数据到剪贴板

## 📝 注意事项

### 数据存储
- 所有数据存储在用户设备本地
- 不上传到服务器
- 不收集用户信息
- 符合隐私保护要求

### AI 解析功能
- 使用 Supabase Edge Function 调用百度文心一言
- 如果服务不可用，自动降级到本地解析
- 不影响核心功能使用

### 微信小程序限制
- 本地存储限制：10MB
- 不支持真实音频播放（使用震动替代）
- 需要配置隐私政策

## 🐛 常见问题

### Q: 如何在手机上预览？
A: 在微信开发者工具中点击"预览"，用微信扫描二维码即可。

### Q: 没有 AppID 可以使用吗？
A: 可以！选择"测试号"即可预览和调试。

### Q: AI 解析失败怎么办？
A: 自动切换到本地解析，或手动点击"切换到本地解析"。

### Q: 数据会丢失吗？
A: 数据存储在本地，除非清除小程序数据，否则不会丢失。

### Q: 可以导出会议记录吗？
A: 可以！在统计页面点击"复制"按钮，复制到剪贴板后粘贴到其他应用。
